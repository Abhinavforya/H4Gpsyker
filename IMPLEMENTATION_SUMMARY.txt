🚀 HOW TO USE

1. Connect Spotify
   └─ Click "Connect Spotify" button
   └─ Authorize with your account
   └─ See your playlists

2. Select Playlist
   └─ Click any playlist card
   └─ Modal opens with songs

3. Search & Select Songs
   └─ Type in search box
   └─ Click checkboxes or use bulk buttons
   └─ Select desired songs

4. Convert to ASCII Art
   └─ Click "Convert Selected Songs"
   └─ Results appear in ASCII & Canvas sections

🔧 CONFIGURATION

Environment Variables (.env):
  SPOTIFY_CLIENT_ID=your_id
  SPOTIFY_CLIENT_SECRET=your_secret
  REDIRECT_URI=http://localhost:8000/auth/spotify/callback
  SESSION_SECRET=your_session_secret

Start Server:
  npm start
  
Access Application:
  http://localhost:8000
