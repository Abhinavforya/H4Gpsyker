# ASCII Framer — minimal frontend

A tiny React + Framer Motion demo that renders animated ASCII art.

Quick start:

```bash
cd ascii-framer
npm install
npm run dev
```

Open: http://localhost:5173

Notes:
- This is a sketch/demo; the `AsciiArt` generator is a placeholder that maps input text into repeated lines.
- Replace the generator with a MIDI/audio-to-ASCII pipeline to integrate with H4G.
